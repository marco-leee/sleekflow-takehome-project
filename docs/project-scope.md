# Final Project Scope

[← Back to documentation index](main.md)

## In Scope

1. TODO list management
   1. CRUD operations
   2. Filtering and sorting
   3. Recurring tasks
   4. Concurrent update 

## Out of Scope

1. Authentication
2. Permission management
3. Logging and auditing
4. Infrastructure and deployment
5. Implementation of industry security standards
6. Integration with existing systems and processes
7. Security
8. DB replication and backup

9. Scope Notes (for MVP)
   1. Prioritize correctness of CRUD, dependencies, recurrence generation, filtering/sorting, and performance for large lists.
   2. Defer “nice-to-haves” like real-time sync, bulk actions, and full auth if time is limited.
